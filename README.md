Simulador de Planeta Tierra

REQUIERE:
======

* python3 
* Módulos requeridos `turtle`, `time`, `random`.
para la instalación use `pip3`
EJEMPLO:
`pip3 install turtle` 

DESCARGAR:
======

```
git clone
```

USAR:
======

* `cd PlanetEarthS/`
* `chmod +x PlanetEarthS.py`
* `chmod +x requirements.txt`
* `pip3 install -r requirements.txt`
* `python3 PlanetEarthS.py`
